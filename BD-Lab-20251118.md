# Cos'è una vista

Una vista è una tabella virtuale, i cui dati non sono davvero memorizzati su disco. è una visualizzazione dei dati di altre tabelle, è definita da una query a partire da altre tabelle.

Per cosa è utile? Magari a uno vuoi dare il permesso di vedere una parte di una tabella e non tutta, di base dai il permesso per una tabella intera, con una vista tratti una query come tabella e dai permessi per quella.
